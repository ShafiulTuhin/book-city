##Project Name: 'Book-City'

##Purpose:
--> Book-City is a Next.js-based web application where users can explore a collection of books and borrow them through a simple and user-friendly interface.
--> Authentication & Authorization implemented using normal sign-in and Google sign-in, ensuring secure and smooth user access management.
--> Learning-focused project: This application is built as a practical learning experience to strengthen my understanding of Next.js, authentication systems, and real-world full-stack development workflows.

##live URL : https://book-city.vercel.app

##key features:
--> Extensive Book Collection – Browse a variety of books across different categories with detailed information for each book.
--> User Profile Management – Users can manage their profile and view their activity within the platform.
--> Secure Authentication System – Supports both email/password login and Google sign-in for a smooth and secure user experience.
